Projects: Personal, Research, and Academic

Neural Spike Sorting (Current)

Working on optimizing neural spike detection algorithms, using MatLab and implementing Franke's Bayes-optimal template matching algorithm. Current objectives include improving upon Franke's linear discriminant analysis method for developing a filter. Challenges include finding efficient means of analysis for the very large data set.
Tensor Flow (Current)

Using Tensor Flow to construct a neural network for a neural spike classification task. A side project based off my research project in the Feldman Lab.
Psiturk Behavioral Experiment App

Implemented the front-end, back-end, and data processing code of a cognitive science experiment testing the ability of the brain to abstract rules. Used HTML, CSS, and Javascript. The app was created on top of Amazon's Mechanical Turk platform and hosted on OpenShift.
Pacman AI algorithms (Academic)

Developed AI for playing Pacman in various scenarios. Implemented basic AI concepts such as reinforcement learning, Baye's Net Inference, and particle filtering. GUI and game logic provided by course staff.
